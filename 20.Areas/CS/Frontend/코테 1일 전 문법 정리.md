## 자주 쓰는 문법들

### 1. 배열(Array) 조작

- **정렬 (`sort`)**: 
  `arr.sort((a, b) => a - b)` (오름차순), 
  `arr.sort((a, b) => b - a)` (내림차순). 
  콜백 없으면 **사전순(문자열)** 정렬되니 숫자 정렬 시 주의.
    
- **2차원 배열 초기화**: 
  `const arr = Array.from({ length: N }, () => Array(M).fill(0))`
    
- **중복 제거**: `const unique = [...new Set(arr)]`
    
- **배열 합계**: `arr.reduce((acc, cur) => acc + cur, 0)`
    

### 2. 자료구조 (Map, Set)

#### 1) Map (키-값 쌍)

- **추가/수정**: `map.set(key, value)`
- **조회**: `map.get(key)` (값 반환, 없으면 `undefined`)
- **존재 확인**: `map.has(key)` (`true` / `false`)
- **삭제 (하나만)**: `map.delete(key)`
- **삭제 (전체)**: `map.clear()`
- **크기 확인**: `map.size` (배열의 `length`와 같은 역할)

#### 2) Set (중복 없는 값들의 집합)

- **추가**: `set.add(value)`
- **조회**:  직접 값을 뽑아야 한다면 배열로 변환: `const arr = [...set]` 후 인덱스 접근.
- **존재 확인**: `set.has(value)` (`true` / `false`)
- **삭제 (하나만)**: `set.delete(value)`
- **삭제 (전체)**: `set.clear()`
- **크기 확인**: `set.size`

### 3. 문자열(String) 처리

- **반복**: `'abc'.repeat(3)`
    
- **패딩**: `str.padStart(length, '0')`, `str.padEnd(length, '0')` (시간/날짜 포맷팅에 유용).
    
- **교체**: `str.replace(/pattern/g, 'replacement')` (전체 교체 시 정규식 `g` 플래그 필수).
    

### 4. 수학 & 숫자 (Math)

- **소수점 처리**: `Math.floor()` (내림), `Math.ceil()` (올림), `Math.round()` (반올림).
    
- **정수 변환**: `Math.trunc()` (소수점 버림), `parseInt(str)` (진법 변환 시 유용).
    
- **큰 수 처리**: $2^{53}-1$ 넘어가면 `BigInt` 사용 (`10n`, `BigInt(s)`).
    

### 5. 알고리즘 필수 패턴

- **BFS**: `while`문과 `queue` 사용. `shift()`는 $O(N)$이니 데이터 크면 인덱스(head)로 관리할 것.
    ```js
    function bfs(start) {
	  const queue = [start];
	  visited[start] = true;
	  while (queue.length) {
	    const curr = queue.shift();
	    for (const next of graph[curr]) {
	      if (!visited[next]) {
	        visited[next] = true;
	        queue.push(next);
	      }
	    }
	  }
	}
    ```
- **DFS**: 재귀 사용. `stackTraceLimit` 설정 확인하거나 반복문으로 구현.
	```js
	function dfs(curr) {
	  visited[curr] = true;
	  for (const next of graph[curr]) {
	    if (!visited[next]) dfs(next);
	  }
	}
	```
- **조합/순열**: 재귀와 `visited` 배열 활용 템플릿 머릿속에 넣어둘 것.
    
- **이진 탐색**: `while (left <= right)` 기반 중간값 비교 패턴.
    

### 6. 디버깅 및 팁

- **구조 분해 할당**: `[a, b] = [b, a]` (값 교환), `const [first, ...rest] = arr`.
    
- **단축 평가**: `const val = str || 'default'` (Falsy값 처리).
    
- **조건문**: 복잡하면 `switch`나 `Object` 매핑 활용.

## 자주 쓰는 매서드

### 1. 배열 (Array)

- **추가/제거**:
    
    - `push()`, `pop()`: 뒤에서 넣고 빼기
        
    - `unshift()`, `shift()`: 앞에서 넣고 빼기 (**`shift`는 $O(N)$이라 큐 구현 시 주의**)
        
- **고차 함수**:
    
    - `map()`: 각 요소 변경해서 새 배열 반환
        
    - `filter()`: 조건 맞는 요소만 추려서 새 배열 반환
        
    - `reduce((acc, cur) => ..., init)`: 누적합이나 객체 변환 시 필수
        
    - `forEach()`: 단순 순회
        
- **자르고 붙이기**:
    
    - `slice(start, end)`: **원본 보존**, 구간 복사
        
    - `splice(start, count, add)`: **원본 수정**, 특정 요소 제거/추가
        
    - `join('')`: 배열을 문자열로 합치기
        
    - `concat()`: 배열 합치기
        
- **찾기**:
    
    - `indexOf(val)`, `lastIndexOf(val)`: 인덱스 찾기
        
    - `includes(val)`: 존재 여부 확인 ($O(N)$)
        
    - `find()`, `findIndex()`: 조건 맞는 첫 요소/인덱스 찾기
        

### 2. 문자열 (String)

- **자르기**:
    
    - `split('')`: 문자열을 배열로 (분할 정복/가공 필수)
        
    - `slice(start, end)`: 특정 구간 추출
        
    - `substring(start, end)`: `slice`와 유사
        
- **변환**:
    
    - `toLowerCase()`, `toUpperCase()`: 대소문자 변환
        
    - `trim()`: 앞뒤 공백 제거 (입력값 처리 필수)
        
    - `replace(target, replacement)`: 첫 번째만 바뀜 (전체는 `/reg/g`나 `replaceAll` 사용)
        
- **기타**:
    
    - `repeat(n)`: 문자열 $n$번 반복
        
    - `padStart(len, '0')`, `padEnd()`: 길이 맞추기
        

### 3. 수학 (Math) - 숫자 처리

- `Math.abs()`: 절대값
    
- `Math.min()`, `Math.max()`: 최소/최대 (**배열 넣을 땐 `...` 전개 연산자 필수**)
    
- `Math.floor()`: 내림 (음수 주의)
    
- `Math.ceil()`: 올림
    
- `Math.round()`: 반올림
    
- `Math.sqrt()`: 제곱근
    
- `Math.pow(a, b)`: $a^b$ ($a$**$b$와 같음)
    

### 4. 기타 필수

- **Object**:
    
    - `Object.keys(obj)`: 키 배열 반환
        
    - `Object.values(obj)`: 값 배열 반환
        
    - `Object.entries(obj)`: `[키, 값]` 쌍 배열 반환
        
- **전역 함수**:
    
    - `parseInt(str, radix)`: 문자열을 정수로 (2진법 -> 10진법 등 유용)
        
    - `Number()` / `+str`: 숫자로 형변환
        
    - `.toString(radix)`: 숫자를 $n$진법 문자열로 변환
