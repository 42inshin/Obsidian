### 1. 배열(Array) 조작

- **정렬 (`sort`)**: 
  `arr.sort((a, b) => a - b)` (오름차순), 
  `arr.sort((a, b) => b - a)` (내림차순). 
  콜백 없으면 **사전순(문자열)** 정렬되니 숫자 정렬 시 주의.
    
- **2차원 배열 초기화**: 
  `const arr = Array.from({ length: N }, () => Array(M).fill(0))`
    
- **중복 제거**: `const unique = [...new Set(arr)]`
    
- **배열 합계**: `arr.reduce((acc, cur) => acc + cur, 0)`
    

### 2. 자료구조 (Map, Set)

#### 1) Map (키-값 쌍)

- **추가/수정**: `map.set(key, value)`
- **조회**: `map.get(key)` (값 반환, 없으면 `undefined`)
- **존재 확인**: `map.has(key)` (`true` / `false`)
- **삭제 (하나만)**: `map.delete(key)`
- **삭제 (전체)**: `map.clear()`
- **크기 확인**: `map.size` (배열의 `length`와 같은 역할)

#### 2) Set (중복 없는 값들의 집합)

- **추가**: `set.add(value)`
- **조회**:  직접 값을 뽑아야 한다면 배열로 변환: `const arr = [...set]` 후 인덱스 접근.
- **존재 확인**: `set.has(value)` (`true` / `false`)
- **삭제 (하나만)**: `set.delete(value)`
- **삭제 (전체)**: `set.clear()`
- **크기 확인**: `set.size`

### 3. 문자열(String) 처리

- **반복**: `'abc'.repeat(3)`
    
- **패딩**: `str.padStart(length, '0')`, `str.padEnd(length, '0')` (시간/날짜 포맷팅에 유용).
    
- **교체**: `str.replace(/pattern/g, 'replacement')` (전체 교체 시 정규식 `g` 플래그 필수).
    

### 4. 수학 & 숫자 (Math)

- **소수점 처리**: `Math.floor()` (내림), `Math.ceil()` (올림), `Math.round()` (반올림).
    
- **정수 변환**: `Math.trunc()` (소수점 버림), `parseInt(str)` (진법 변환 시 유용).
    
- **큰 수 처리**: $2^{53}-1$ 넘어가면 `BigInt` 사용 (`10n`, `BigInt(s)`).
    

### 5. 알고리즘 필수 패턴

- **BFS**: `while`문과 `queue` 사용. `shift()`는 $O(N)$이니 데이터 크면 인덱스(head)로 관리할 것.
    
- **DFS**: 재귀 사용. `stackTraceLimit` 설정 확인하거나 반복문으로 구현.
    
- **조합/순열**: 재귀와 `visited` 배열 활용 템플릿 머릿속에 넣어둘 것.
    
- **이진 탐색**: `while (left <= right)` 기반 중간값 비교 패턴.
    

### 6. 디버깅 및 팁

- **구조 분해 할당**: `[a, b] = [b, a]` (값 교환), `const [first, ...rest] = arr`.
    
- **단축 평가**: `const val = str || 'default'` (Falsy값 처리).
    
- **조건문**: 복잡하면 `switch`나 `Object` 매핑 활용.
