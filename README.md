# 🧠 My Digital Brain (Obsidian Vault)

개인적인 학습 내용과 지식 파편들을 체계적으로 정리하고 백업하기 위한 **Obsidian Vault**입니다.

---

## 📂 저장소 구조

기본적으로 [PARA 프로젝트 관리법](https://fortelabs.com/blog/para/)을 응용하여 폴더를 구성하고 있습니다.

* **00. Inbox**: 정리되지 않은 메모, 빠른 기록
* **10. Projects**: 마감 기한이 있는 구체적인 프로젝트 (예: 자격증 공부, 특정 개발 프로젝트)
* **20. Areas**: 지속적인 관심사 및 학습 (예: JavaScript, 알고리즘, CS 지식)
* **30. Resources**: 참고용 자료, 아카이빙 (예: 코드 스니펫, 유용한 링크)
* **40. Archives**: 완료된 프로젝트 또는 더 이상 업데이트하지 않는 자료
* **99. Meta**: 템플릿, 첨부 파일(Images), 설정 파일

---

## 🛠 사용 중인 도구 & 환경

* **Editor**: [Obsidian](https://obsidian.md/)
* **Sync**: GitHub (Obsidian Git 플러그인 사용)
* **Key Plugins**:
    * `Obsidian Git`: 자동 백업 및 푸시
    * `Dataview`: 데이터 쿼리 및 대시보드 생성
    * `Templater`: 문서 구조 자동화
    * `Excalidraw`: 핸드 드로잉 및 다이어그램

---

## 🚀 백업 규칙 (Workflow)

1.  **매일 종료 시**: `Obsidian Git` 플러그인을 통해 자동으로 Commit & Push를 수행합니다.
2.  **커밋 메시지 규칙**:
    * `feat`: 새로운 노트 추가
    * `fix`: 오타 수정 및 내용 보완
    * `docs`: 문서 구조 변경 (템플릿 등)
3.  **이미지 관리**: 모든 이미지는 `99. Meta/Attachments` 폴더에 모아 관리하며, 가급적 WebP 형식을 권장합니다.
